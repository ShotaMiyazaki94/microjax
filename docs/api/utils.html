microjax.utils.match_points(a, b)

Iterate over elements of a in order and find the index of the closest element in b in distance. Return indices which permute b.

This algorithm is not guaranteed to find a permutation of b which minimizes the sum of elementwise distances between every element of a and b (linear sum assignment problem).

Parameters

  • a (array_like) – 1D array of complex numbers.

  • b (array_like) – 1D array of complex numbers.

Returns

Indices which specify a permutation of b.

Return type

array_like

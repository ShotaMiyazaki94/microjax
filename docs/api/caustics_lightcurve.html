Caustics Lightcurve

microjax.caustics.lightcurve.magnifications(w_points, rho, nlenses=2, npts_limb=200, limb_darkening=False, u1=0.0, npts_ld=100, **params)

Extended-source magnification for one to three point lenses.

This helper evaluates the finite-source magnification at each complex position in w_points. It selects between microjax.multipole.mag_hexadecapole() and microjax.caustics.extended_source.mag_extended_source() on a per-point basis, following the original caustics heuristics but adapted to microJAX’s coordinate handling and JAX-backed solvers.

w_points are given in the center-of-mass frame of the first two lenses (or the single lens when nlenses == 1). Internally the coordinates are shifted to the mid-point frame required by the point-source polynomial solver and by mag_extended_source() and shifted back afterwards.

Behaviour summary

  • nlenses == 1: the multipole branch is exact and is used everywhere.

  • nlenses == 2: proximity and planetary-caustic tests determine where the fast multipole estimate is sufficient; otherwise a full contour integration is triggered.

  • nlenses == 3: the ghost-image test is not yet implemented, so the routine falls back to contour integration for every point.

param w_points

Source positions (center-of-mass frame) at which to evaluate the magnification.

type w_points

array_like

param rho

Source radius in Einstein units.

type rho

float

param nlenses

Number of lenses in the system (1, 2, or 3). Defaults to 2.

type nlenses

int, optional

param npts_limb

Base number of sampling points placed uniformly on the source limb before adaptive refinement during contour integrations. Defaults to 200.

type npts_limb

int, optional

param limb_darkening

If True, evaluate the magnification of a linearly limb-darkened source. Defaults to False.

type limb_darkening

bool, optional

param u1

Linear limb-darkening coefficient used when limb_darkening is set. Defaults to 0.0.

type u1

float, optional

param npts_ld

Number of quadrature samples for the Dominik (1998) P and Q integrals when limb darkening is enabled. Defaults to 100.

type npts_ld

int, optional

param **params

Lens configuration parameters forwarded to the underlying solvers:

  • nlenses == 1: no additional keywords are required.

  • nlenses == 2: s (separation) and q (mass ratio m2/m1).

  • nlenses == 3: s, q, q3 (m3/m1), r3 (magnitude of the third lens position), and psi (azimuth of the third lens).

Extra keyword arguments understood by mag_hexadecapole(), mag_extended_source(), or microjax.point_source._images_point_source() (for example custom root-solver settings) are propagated unchanged.

returns

Magnification evaluated at each entry of w_points.

rtype

jax.Array

Notes

Enabling limb darkening typically incurs an order-of-magnitude slowdown because additional quadrature is required. Triple-lens configurations always take the contour-integration path, which is slower but currently the only option with reliable accuracy.

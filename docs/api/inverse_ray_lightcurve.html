Inverse-Ray Lightcurve

Adaptive lightcurve computation mixing multipole and inverse-ray methods.

This module computes microlensing lightcurves by defaulting to the fast hexadecapole approximation and selectively invoking the full inverse-ray finite-source integration near caustics or when required by accuracy tests.

microjax.inverse_ray.lightcurve.mag_binary(w_points: Array, rho: float, r_resolution: int = 1000, th_resolution: int = 1000, u1: float = 0.0, delta_c: float = 0.01, Nlimb: int = 500, bins_r: int = 50, bins_th: int = 120, margin_r: float = 1.0, margin_th: float = 1.0, MAX_FULL_CALLS: int = 500, chunk_size: int = 100, **params) → Array

Binary-lens lightcurve with adaptive full-solve selection.

Parameters follow the underlying inverse-ray integrators. The selection uses a proximity test to caustics plus planetary-caustic checks for small mass ratios. Returns magnifications per input time/position.
